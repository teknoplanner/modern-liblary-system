# Sistem Perpustakaan (Library Management System)
Sistem perpustakaan sederhana berbasis Laravel 10/11 dengan styling Tailwind CSS.

### Fitur:
1. Registrasi Anggota (Pendaftaran anggota baru).
2. Pendaftaran Buku (Manajemen koleksi buku).
3. Transaksi Peminjaman (Meminjam buku untuk anggota).
4. Daftar Pinjaman & Pengembalian (Update status & list transaksi).
5. Laporan Sederhana (Daftar buku yang masih dipinjam).
6. Sistem Login.

### Setup (Instructions for User):
1. Pastikan PHP dan Composer terinstall.
2. Jalankan `composer install && npm install`.
3. Rename `.env.example` ke `.env` dan konfigurasikan database.
4. Jalankan `php artisan migrate`.
5. Jalankan `php artisan serve`.
6. Jalankan `npm run dev` untuk Tailwind.
